# Camunda BPM Team Blog

Generated by [Hugo](hugo) and styled with some fancy [node.js](nodejs) tools like [Grunt](grunt), [less](less).

## Redaction

```sh
hugo -w server
```

## Themeing

```sh
grunt
```

and in a other terminal

```sh
hugo -w server
```
